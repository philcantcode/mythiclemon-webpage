Feature Details

Nanite Landscapes (UE 5.1+)

Requirements: UE 5.1 or later, DX12/Vulkan, SM6
Benefits: Automatic LOD, massive triangle counts, better performance
Limitations: No tessellation, no runtime deformation

PCG - Procedural Content Generation (UE 5.2+)

Requirements: UE 5.2 or later for production use
Benefits: Node-based procedural placement, biomes, runtime generation
Best With: World Partition, Nanite meshes

Substrate Materials (UE 5.4+)

Requirements: UE 5.4 or later for stable production
Benefits: Multi-layer materials, better light interaction, more realistic
Migration: Can mix with legacy materials during transition

Motion Matching (UE 5.2+)

Requirements: UE 5.2 or later for production use
Benefits: Natural character movement, reduced animation work
Requires: Large animation datasets, preprocessing time
